# W4RO

W4RO is an autonomous risk oracle designed for the agent economy.

It transforms raw market signals into structured risk intelligence that autonomous agents can use before executing actions.

## Core Outputs

The oracle produces structured risk data including:

- regime
- risk_score
- leverage_cap
- circuit_breaker_state
- confidence

## Example Output

{
  "version": "1.0.0",
  "symbol": "BTCUSDT",
  "regime": "range",
  "risk_score": 43,
  "stress_index": 0.43,
  "leverage_cap": 3.0,
  "confidence": 0.5
}
